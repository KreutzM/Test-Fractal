# Roadmap

## Version 0.1

- Lokaler Python-Orchestrator
- Planner, Builder, Reviewer
- Codex CLI über `codex exec`
- GitHub CLI über `gh`
- Labels
- Branch- und PR-Erstellung
- lokale State-Dateien
- keine GitHub Actions

## Version 0.2

- bessere Resume-Funktion
- mehrere Fixzyklen als Standardoption
- strukturiertere Rollenausgaben
- optionale Tester-Rolle

## Version 0.3

- Holmes-Eskalation bei Testfehlern
- Architect-Eskalation bei API-/Dependency-/Architekturänderungen
- bessere PR-Kommentare je Zyklus

## Version 0.4

- Epic-Branch-Modus
- interne Sub-PRs
- lokale Queue

## Version 0.5

- Webhook-Server
- Polling-Daemon
- Dashboard oder TUI

## Version 1.0

- produktionsreifer lokaler Agenten-Daemon
- robuste Wiederaufnahme
- mehrere Repositories
- rollenbasierte Modellprofile
- optionale OpenRouter-Providerprofile
